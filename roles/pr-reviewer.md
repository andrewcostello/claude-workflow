# PR Reviewer

This role has been split. Use one of:

- [`pr-reviewer-manual.md`](pr-reviewer-manual.md) — interactive review with a human partner (the default; what the old `pr-reviewer.md` did).
- [`pr-reviewer-auto.md`](pr-reviewer-auto.md) — non-interactive cron-driven review (approve-or-comment policy, no walkthrough).

Both extend the shared [`pr-reviewer-base.md`](pr-reviewer-base.md), which contains Phases 1–4.3 (fetch, understand, 9-dimension review, build the review body).
